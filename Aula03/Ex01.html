Destaques no texto

Podemos realizar destaques no texto, tais como:
negrito,
texto destacado,
itálico,
texto com ênfase. :)

A diferença entre a tag < b > que faz negrito para a tag < strong > é que a tag strong dará um destaque semântico, o mesmo é válido para a tag < em >.
